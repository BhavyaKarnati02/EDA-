# Exploratory Data Analysis Project

This project analyzes a layoffs dataset using statistical summaries, visualizations, and correlation analysis to identify trends and patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Outputs
- Distribution of Layoffs
- Correlation Heatmap
- Top Companies Analysis
- Top Countries Analysis

## Outcome
Developed analytical thinking and data exploration skills through exploratory data analysis.
